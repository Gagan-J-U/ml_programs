# Machine Learning Lab Programs

## Overview

This repository contains Machine Learning Lab programs implemented in Python. The programs cover basic Python operations, search algorithms, game-playing algorithms, classification techniques, clustering methods, dimensionality reduction techniques, and neural networks.

---

## Programs

### 1. Data Structure Operations

Write a simple Python program using conditional statements and looping to perform:

* Insert
* Update
* Delete
* Display
* Sorting
* Searching

on the following data structures:

* List
* Tuple
* Set
* Dictionary

---

### 2. Data Visualization

Visualize n-dimensional data using:

* Scatter Plot
* Box Plot
* Heat Map
* Contour Plot
* 3D Surface Plot

using Python libraries such as Pandas, Matplotlib, and Seaborn.

---

### 3. Hill Climbing Algorithm

Implement the Hill Climbing Algorithm to find the optimal solution for a given objective function.

Implement the A* Search Algorithm.

## Prerequisites

* Python 3.8 or above
* Visual Studio Code
* Required Python Libraries

Install the required libraries using:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

---

## How to Run the Programs in VS Code

### Step 1: Clone the Repository

```bash
git clone <repository-url>
cd ML-Lab
```

### Step 2: Open in VS Code

```bash
code .
```

or open the project folder manually in Visual Studio Code.

### Step 3: Install Required Libraries

Open Terminal in VS Code and run:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

### Step 4: Run a Program

Example:

```bash
python hill_climbing.py
```

Similarly:

```bash
python best_first_search.py
python astar.py
python minimax.py
python alphabeta.py
python iris_naive_bayes.py
python titanic_naive_bayes.py
```
